# 🚀 33.8M+ Instacart Data Optimization ($0 Cloud Cost)

## 📌 Overview
An end-to-end data processing project executing **33.8 million transaction records** on a constrained local setup without any external cloud infrastructure.

## 🛠️ System Constraints vs Solution
- **Hardware:** Core i3, 4GB RAM Local PC
- **Infrastructure Cost:** $0 (Zero Cloud OpEx)
- **Core Technique:** R In-Memory Downcasting & Byte-Level Optimization

## 📊 Key Achievements
1. Reduced memory footprint by **80%+** using precise data typing (`as.integer`, `as.numeric`).
2. Cleansed and aggregated **33.8M rows** without triggering Out-Of-Memory (OOM) errors.
3. Generated clean dataset ready for Tableau business visualizations.
